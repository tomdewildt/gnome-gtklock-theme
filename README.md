Simple GTKLock theme inspired by [GNOME](https://www.gnome.org/).

# How To Install

Prerequisites:
* gtklock version ```4.0.0``` or later

### Install

1. Run ```make install``` to install the theme.

### Uninstall

1. Run ```make uninstall``` to uninstall the theme.
